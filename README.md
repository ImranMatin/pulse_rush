# Pulse-Rush
Human Factors / Computer Science 4326/5326


We need a personal computer/laptop OS that can run Python Code.
Visual Display Monitor and Keyboard for input.
We need C++ to program the microcontroller.
We need to have the Python installed to run the game.
Also we need Pygame library installed using pip install pygame.
We need PySerial package installed so that we can communicate with external hardware by pip install pyserial.
We need a heart rate sensor connected to the Arduino microcontroller, LCD display and breadboard.
Then connect the hardware to the laptop or PC. 

For HR/BPM sensor we need to have 
Arduino UNO or another Arduino microcontroller
Pulse Sensor
LED
220-ohm resistor
L2C module
Jumper Wires and Breadboard
USB cable to upload code to Arduino
LCD Screen

We can run the game by using python main.py in the terminal in the project directory
Then the main menu of the game will be displayed. 
We hit enter to start the game. 
We press the space button to shoot. 
We use left, right, up, down in the keyboard to make the player move and avoid enemies.
The players heart rate will be shown in the LCD as well as the computer or a laptop screen.
